# Hybrid-Quantum Neural Architecture Search for The Proximal Policy Optimization Algorithm
An implementation of Regularized Evolution, a Neural Architecture Search Algorithm, to search for an optimal configuration of classical and quantum layers in a Hybrid Classical-Quantum Neural Network
, trained using the Proximal Policy Optimization Algorithm in a Reinforcement Learning Environement.


arxiv: https://arxiv.org/abs/2501.10673
